### Project Overview
This repository contains a fully functional, single-page, responsive, and interactive company website for the fictional sustainable coffee brand **EcoBrew Coffee**.  
The website serves as the **only submission deliverable** (as per assignment guidelines) and comprehensively covers every required element:

1. Company creation (overview, story, vision, mission)  
2. Products, pricing strategy, and financial highlights  
3. Target market analysis (demographics, psychographics, geographics)  
4. Market size and competitive positioning  
5. Detailed Integrated Marketing Communications (IMC) Plan  
   - Advertising (Digital, TV, Outdoor)  
   - Sales Promotion  
   - Public Relations & Events  
   - Social & Mobile Marketing  
   - Personal Selling  
   - Justification for tools not used  
6. Complete budget breakdown with realistic cost estimates and ROI projections  
7. Persuasive pitch deck section for senior management budget approval  

---

### Website Features
- 100% standalone – **single HTML file** (no backend required)  
- Fully responsive (mobile, tablet, desktop)  
- Smooth scrolling navigation with sticky navbar  
- Scroll animations (AOS library)  
- Interactive charts using Chart.js:  
  - Revenue growth bar chart  
  - Margin doughnut chart  
  - Market size line chart  
  - Competitive positioning scatter chart  
  - Budget allocation pie chart  
- Professional branding with consistent color palette, typography, and high-quality imagery  
- Simulated e-commerce elements (Add to Cart, Subscribe)  
- Executive pitch section with call-to-action
